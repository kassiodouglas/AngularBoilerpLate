Documentação do Módulo Formulary

Visão Geral

Este módulo fornece diversos componentes de formulários, incluindo entradas, validadores e utilitários para melhorar o manuseio de formulários em aplicações Angular.

Componentes

  • Datepickery: Um componente de seleção de data.
  • Debugy: Ajuda a depurar os valores do formulário.
  • Inputy: Componente de entrada personalizado.
  • Selecty: Componente de seleção suspensa.
  • Signatury: Campo para captura de assinatura.
  • Switchy: Componente de alternância.
  • Textareay: Área de texto personalizada.

Validação

O módulo inclui vários validadores:

  • DateAfterNowValidator: Garante que a data seja posterior à data atual.
  • ExactLengthValidator: Valida um comprimento exato de caracteres.
  • OnlyLetters: Restringe a entrada para apenas letras.
  • OnlyLettersNumbers: Permite apenas caracteres alfanuméricos.
  • OnlyNumbers: Restringe a entrada para apenas números.

Como Usar

Para utilizar este módulo em seu projeto Angular, importe o FormularyModule no seu módulo principal.

import {{ '{' }} FormularyModule {{ '}' }} from './formulary/formulary.module';
NgModule({{ '{' }}
imports: [FormularyModule],
{{ '}' }})
export class AppModule {{ '}' }}

Exemplo de Uso dos Componentes

Você pode utilizar os componentes nos seus templates da seguinte maneira:

<inputy label="Nome" formControlName="nome"></inputy>
<selecty label="Escolha uma opção" formControlName="opcao" [options]="[]"></selecty>
<switchy label="Ativar" formControlName="ativo"></switchy>
<textareay label="Descrição" formControlName="descricao"></textareay>
<datepickery label="Data de Nascimento" formControlName="data"></datepickery>
